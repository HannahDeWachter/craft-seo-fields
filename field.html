          # Field

          # ⚠️ Important ⚠️

          The plugin assumes that the handle of the SEO Fields field will be simply: seo. If it can't find that field, it won't output anything.

          If you use a different handle for your field, simply copy this file (opens new window) to the config directory of your project and change the fieldhandle to your own.
